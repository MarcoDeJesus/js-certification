The if...else statement

If statement test if the condition(s) exist
Single condition
if (expression) {
    //statements to execute if expression is true
}
Multiple conditions
if (expression) {
    //statements to execute if expression is true
} 
else {
    //statements to execute if expression is false
}
Additional if conditions
if (condition a) {
    //statements to execute
} else if (condition b) {
    //statements to execute
} else if (condition c) {
    //statements to execute
} else {
    //statements to execute
}

The while Statement


The while statement is used to execute a block of code for as long as (while) a certain test condition is true.

while (expression) {
    //one or more statements to execute
}
Multiple conditions
while ((condition1) || (condition2)) {
    //statements to execute
}

The while Statement


The do...while statement operates like the while statement, with one key difference: The do...while statement does not check the conditional expression until after the first time through the loop, guaranteeing that the code within the curly braces will execute at least once.

do {
    //statements to execute
} while (conditional expression)

The for statement


The primary purpose of the for statement is to repeat a group of statements for some particular range of values.

  • Statement 1 is executed (one time) before the execution of the code block.
  • Statement 2 defines the condition for executing the code block.
  • Statement 3 is executed (every time) after the code block has been executed.

for (statement 1; statement 2; statement 3) {
    // code block to be executed
}

The break Statement


A way to exit a loop that would otherwise continue to execute is to use the break statement.

You can use the break statement within the for, while and do...while loops, as well as with the switch statement.

for(var i = 0; i <= 10; i++){
    document.write("Loop number: " + i);
    if(i == 4) break;
}

The continue statement


The continue statement is used to force the flow of control back to the top of a loop.

The continue statement can be used only within a for or a while loop.

while (curYear <= 2100) {
    curYear++;
    
    if (curYear % 4) continue;
    
    leapYears++;
    document.write(curYear + " ");
    
}

The switch statement


The switch statement compares a value against other values, searching for a match. If a match is found, then the code associated with the match will execute. The break statement is then used to exit the switch block of code.

switch (expression) {
    case "str1" :
        //statements to execute
    break;
    
    case "str2" :
        //statements to execute
    break;
    
    case "str3" :
        //statements to execute
    break;
    
    default :
        //statements to execute
    break;
}
